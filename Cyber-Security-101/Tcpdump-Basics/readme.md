# Tcpdump: The Basics (TryHackMe)

## 🧠 Learnings
- Tcpdump captures network packets from a network interface
- It helps analyze traffic in real time
- Useful for troubleshooting and security monitoring

## 🔧 Commands Used
tcpdump -i eth0
tcpdump port 80
tcpdump -w capture.pcap

## 📌 Key Concepts
- Packet capture
- Network interfaces
- Traffic filtering
